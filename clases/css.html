Diseño, Jerarquías y Ecosistema de Trabajo

1. Especificidad: La Lucha por el Estilo

Escala de Pesos (Puntaje Acumulativo):
Etiquetas (p, div) 1 Punto
Clases (.clase) 10 Puntos
IDs (#unico) 100 Puntos
In-line (Atributo style) 1000 Puntos

⚠️ NOTA: El estilo In-line gana técnicamente pero es mala práctica arquitectónica. Usar siempre archivos externos.

Anatomía del Box Model
CONTENT
  • ■ Margin: Espacio fuera del borde.
  • ■ Border: El límite de la caja.
  • ■ Padding: Espacio entre borde y contenido.

2. Aplicación Real: Tarjeta Institucional

Aquí vemos cómo el Padding (espacio interno) genera el aire necesario para que el texto luzca profesional.

HTML (Estructura)
<div class="card-box">
 <p name="lomas-titulo">
  PROGRAMACIÓN
 </p>
</div>
CSS (Reglas)
.card-box {
 background: #003870;
 padding: 40px;
}
p[name^="lomas"] {
 color: #fff;
 font-weight: 900;
}
